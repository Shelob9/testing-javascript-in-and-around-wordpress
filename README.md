## Development

Boilerplate used: https://github.com/whoisryosuke/mdx-deck-netlify

To run the presentation deck in development mode:

```sh
yarn #install
yarn start #start
```

Edit the [`deck.mdx`](deck.mdx) file to get started.

## Deploy To Github Pages

```bash
yarn deploy
git add . "rebuild"
git push
```
